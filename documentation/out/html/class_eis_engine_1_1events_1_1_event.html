Detailed Description

template<typename Owner, typename ... Args>
class EisEngine::events::Event< Owner, Args >


Represents an event that listeners can subscribe to. Events have an owner and an arbitrary number of arguments that are passed to the listener when invoked. Only the owner is allowed to invoke the event.
