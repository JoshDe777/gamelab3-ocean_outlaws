Detailed Description


Contains rendering data for sprites.
Requires a SpriteMesh component for proper use.

Parameters
spritePath- std::filesystem::path: the path from the assets folder to the image to be rendered as a sprite.
color- Color: A 4D vector representing a mesh's color in RGBA%.
layer- std::string: The rendering layer for meshes paired with a renderer.
Currently supported: {"UI" for UI Elements, and [any other string] for regular rendering}.
