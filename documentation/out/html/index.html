Quick Start

This is a simple quick start guide how to clone my wonderful project and get it running.

  1. You can clone the code with this command:
git clone --recursive --depth 1 **LINK TO MY REPOSITORY (to be updated)**
  1. Once cloned, you can open the root folder in CLion. And load the CMAKEList.txt file
  2. In CLion go to Settings->Build,Execution,Deployment->Toolchains add the Microsoft Visual Studio Compiler (2019 or 2022) and hit Apply
  3. You should now have a running configuration -> compile the code and have fun!
